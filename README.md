SimpleBBS
=========

Let BBS post easier to write!

---

### Title & Subtitle
```
head TITLE
subh SUBTITLE
```

### Section
```
section NAME
CONTENTS
section-end
```

### Strike
```
~~TO_STRIKE~~
```

### Bold
```
**TO_BOLD**
```

### Code
<pre>
```LANG
CONTENTS
```
</pre>

### URL
```
(SHOW TEXT)[LINK]
```

### List
```
list
* ITEM
* ITEM
list-end
```

```
list=1
* ITME
* ITEM
list-end
```
